# A tool for plotting 3D functions
![demo gif](static/plotter.gif)
## Controls
|Button|Description|
|---|---|
|Arrows|Rotate view|
|F|Enable/Disable filling surface faces|
|G|Enable/Disable showing coordinate grid|
|+ -|Increase/Decrease # of isolines|
|< >|Increase/Decrease Surface quality|
## How to run
Build CMake, run.
